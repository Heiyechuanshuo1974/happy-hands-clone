# happy-hands-clone

Happy Hands 网站复刻项目。

## 技术栈
- Frontend: Vue 3 + Vite + Vue Router
- Backend: Java 17 + Spring Boot 3 + Spring Security + JPA

## 目录结构
```bash
frontend/   # 前端项目
backend/    # 后端项目
docs/       # 项目文档
```

## 当前进度
- 已完成 Vue 前端基础骨架
- 已完成 Spring Boot 后端基础骨架
- 已完成开发方案文档初版

## 快速开始
### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
mvn spring-boot:run
```

> 注意：当前机器未安装 Java，因此后端尚未本地验证启动。
