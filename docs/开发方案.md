# Happy Hands Clone 开发方案

## 1. 项目目标
复刻目标网站的企业官网 + 商品目录能力，首期聚焦 B2B 展示、商品浏览、询价咨询，不直接进入完整支付电商闭环。

## 2. 技术选型
### 前端
- Vue 3
- Vite
- Vue Router
- Pinia（下一步接入）
- Axios（下一步接入）

### 后端
- Java 17
- Spring Boot 3
- Spring Web
- Spring Security
- Spring Data JPA
- MySQL 8
- SpringDoc OpenAPI

## 6. 启动说明
### 前端
```bash
cd frontend
npm install
npm run dev
```

### 后端
> 当前环境未安装 Java / Maven，骨架已生成，安装 JDK17 与 Maven 后可启动

```bash
cd backend
mvn spring-boot:run
```
